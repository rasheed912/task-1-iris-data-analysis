# Iris Dataset Analysis

## Objective
To explore and visualize the Iris dataset using Python.

## Tools Used
- Python
- pandas
- matplotlib
- seaborn

## Approach
- Loaded dataset using seaborn
- Checked structure and cleaned data
- Performed EDA using scatter plots, histograms, and box plots

## Key Insights
- Strong relationship between petal length and sepal length
- Data distribution is mostly normal
- Clear differences between species

## Conclusion
EDA helped in understanding patterns and relationships in the dataset.
